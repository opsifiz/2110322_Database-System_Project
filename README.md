# 2110322_Database-System_Project

## Campground Booking

Functional Requirement
1. [The system shall allow a user to register by specifying the name, telephone number, email, and password.](/1)
2. [After registration, the user becomes a registered user, and the system shall allow the user to log in to use the system by specifying the email and password. The system shall allow a registered user to log out.](./2)
3. [After login, the system shall allow the registered user to book up to 3 nights by specifying the date and the preferred campground. The campground list is also provided to the user. A campground information includes the campground name, address, and telephone number.](./3)
4. [The system shall allow the registered user to view his campground bookings.](./4+7)
5. [The system shall allow the registered user to edit his campground bookings.](./5+8)
6. [The system shall allow the registered user to delete his campground bookings.](./6+9)
7. [The system shall allow the admin to view any campground bookings.](./4+7)
8. [The system shall allow the admin to edit any campground bookings.](./5+8)
9. [The system shall allow the admin to delete any campground bookings.](./6+9)
